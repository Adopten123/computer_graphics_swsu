## Установка зависимостей (Ubuntu)

Перед запуском убедитесь, что в вашей системе установлены все необходимые библиотеки для работы с графикой. Для этого откройте терминал и выполните следующую команду:

```bash
sudo apt update && sudo apt install build-essential libglew-dev libglfw3-dev libglm-dev
```

## Компиляция и Запуск

Для сборки проекта используйте стандартный компилятор `g++`:

### Компиляция:

```bash
g++ lab_geo.cpp -o lab_geo -lGL -lGLEW -lglfw
```

### Запуск:

```bash
./lab_geo
```

## Интерфейс управления (Hotkeys)

В приложении реализовано управление с клавиатуры для выполнения различных операций.

### Переключение режимов

* **Клавиша 1**: Режим Поверхности Безье (Задание 4).
* **Клавиша 2**: Режим Поверхности Ду-Сабина (Задание 5).

### Параметры разбиения (Задание 6в)

* **Стрелка Вверх (UP)**:

  * **Для Безье**: Увеличивает плотность сетки (параметр разбиения).
  * **Для Ду-Сабина**: Выполняет следующий этап подразделения (расчет новых граней).
* **Стрелка Вниз (DOWN)**:

  * **Для Безье**: Уменьшает плотность сетки.
  * **Для Ду-Сабина**: Сбрасывает к исходному контрольному полигону (кубу).

### Визуализация (Задание 6а, 6б)

* **Клавиша Z**: Включить/выключить отображение контрольных точек (красные для Безье, желтые для Ду-Сабина).
* **Клавиша X**: Включить/выключить отображение каркаса поверхности (Wireframe).

## Описание реализованных алгоритмов

| Алгоритм          | Реализация                                                                                                                |
| ----------------- | ------------------------------------------------------------------------------------------------------------------------- |
| Поверхность Безье | Математический расчет по формуле двойной суммы полиномов Бернштейна. Поддерживает сетку N×M точек.                        |
| Ду-Сабин          | Поэтапное разбиение (subdivision). На каждой итерации вычисляются новые точки грани на основе средних значений координат. |

## Особенности визуализации

* **Авто-поворот**: Сцена плавно вращается вокруг оси Y, что позволяет оценить объемность поверхности без использования мыши.

* **Цветовая индикация**:

  * Бирюзовый — поверхность Безье.
  * Зеленый — поверхность Ду-Сабина.
  * Красный/Желтый — точки контрольных полигонов.
